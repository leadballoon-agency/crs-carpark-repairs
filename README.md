## Quick Start

### Local Development

```bash
# Install dependencies
npm install

# Run local server
npm run dev

# Open http://localhost:3000
```

### Deploy to Vercel

1. Push to GitHub repository
2. Connect repo to Vercel
3. Deploy with one click

Or use Vercel CLI:

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Deploy to production
vercel --prod
```
